# ToggleButton
A modern and customizable toggle button implemented in C++ with Qt framework.

## Features
- Customizable size and appearance
- Smooth animation curves
- Easy integration into C++ projects

## Installation
1. Clone this repository:
   ```sh
   git clone https://github.com/TONI7008/ToggleButton.git
   cd ToggleButton
   ```
2. Include `circularprogressbar.h` and `circularprogressbar.cpp` in your project.
